# Слой 2 — петля «Мария ↔ менеджер завода ↔ клиент» (дизайн, строим ПОСЛЕ переезда)

Правила утверждены Сергеем 12.06.2026. Сейчас лиды на фазе 1 ВЫКЛЮЧЕНЫ (только лог).
Это спецификация на сборку после боевого запуска mpk-doska.ru.

## Принцип разделения данных
- **Заводу уходит ОБЕЗЛИЧЕННАЯ заявка**: изделие, объём, расчёт с калькулятора, суть запроса.
  Контакт клиента (телефон/ник/почта) НА ЗАВОД НЕ ПЕРЕДАЁТСЯ.
- **Контакт клиента остаётся у нас** — это наш актив, завод клиента не уводит.
- КП клиенту перевыставляет САМА Мария по контакту, который клиент оставил.

## Поток
1. Мария ведёт диалог, собирает заявку (item / volume / calc / task) + контакт клиента (хранится у нас).
2. Обезличенная заявка → менеджеру завода (канал — УТОЧНИТЬ: TG-группа / MAX / почта завода).
3. Менеджер смотрит наличие, отвечает: цена, наличие, сроки.
4. Мария получает ответ менеджера → формирует КП → шлёт КЛИЕНТУ на его контакт → ждёт ответ.

## Дожим пропавшего после КП (ритм — СРЕДНЕ)
- Касания: **день 1, день 3, день 7** после отправки КП.
- Тон: «что смутило?» → выяснить истинную причину → отработать возражение → попытаться закрыть.
- Если клиент молчит после 3-го касания → **передать лид Сергею** с пометкой «дальше ты» + историей.

## Немедленная эскалация Сергею (Мария НЕ дожимает сама), когда:
- клиент просит живого человека;
- явное жёсткое «нет» / отказ;
- прошла вся цепочка молчания (день 1/3/7);
- **крупный / оптовый заказ** ИЛИ вопрос вне компетенции Марии — сразу Сергею, не рискуя.
  (Порог «крупный» уточнить вместе с прайсом завода; пока — по объёму/прямому признаку опта.)
